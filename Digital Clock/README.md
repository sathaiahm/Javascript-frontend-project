# Digital Clock

A real-time digital clock that displays the current time in hours, minutes, and seconds.

## Features
- Real-time clock display
- Updates every second
- 12-hour or 24-hour format support
- Clean digital interface

## Files
- `index.html` - HTML structure
- `index.css` - Clock styling
- `index.js` - Clock functionality

## How to Use
1. Open `index.html` in your web browser
2. The clock will automatically display the current time
3. The time updates every second
