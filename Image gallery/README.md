# Image Gallery

An interactive image gallery with navigation buttons to browse through images.

## Features
- Browse images with next/previous buttons
- Display current image with description
- Keyboard navigation support
- Responsive gallery layout

## Files
- `index.html` - HTML structure
- `index.css` - Gallery styling
- `index.js` - Navigation logic
- `image-1.png` to `image-6.png` - Gallery images
- `next.png` - Next button icon
- `back.png` - Previous button icon

## How to Use
1. Open `index.html` in your web browser
2. Click the "next" button to view the next image
3. Click the "back" button to view the previous image
4. Images cycle through the gallery
