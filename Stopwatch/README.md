# Stopwatch

A stopwatch application to measure elapsed time with start, stop, and reset functionality.

## Features
- Start and stop timing
- Reset to zero
- Lap time recording
- Display hours, minutes, seconds, and milliseconds
- Responsive design

## Files
- `index.html` - HTML structure
- `index.css` - Stopwatch styling
- `index.js` - Timer logic

## How to Use
1. Open `index.html` in your web browser
2. Click "Start" to begin the timer
3. Click "Stop" to pause the timer
4. Click "Reset" to clear the timer
5. Click "Lap" to record lap times
