# Hide and Show Password Toggle

A password input component with a toggle button to show or hide the password text.

## Features
- Toggle password visibility
- Eye icon for visual feedback
- Secure password input field
- Click to show/hide password

## Files
- `index.html` - HTML structure
- `index.css` - Input styling
- `index.js` - Toggle functionality
- `eye-open.png` - Icon for showing password
- `eye-close.png` - Icon for hiding password

## How to Use
1. Open `index.html` in your web browser
2. Enter your password in the input field
3. Click the eye icon to toggle password visibility
4. Click again to hide the password
