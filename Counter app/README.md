# Counter App

A simple counter application that allows users to increment, decrement, and reset a counter value.

## Features
- Increment counter
- Decrement counter
- Reset counter to zero
- Display current counter value

## Files
- `index.html` - HTML structure
- `index.css` - Styling
- `index.js` - Counter logic

## How to Use
1. Open `index.html` in your web browser
2. Click the increment button to increase the counter
3. Click the decrement button to decrease the counter
4. Click the reset button to reset the counter to zero
