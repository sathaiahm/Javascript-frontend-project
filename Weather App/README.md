# Weather App

A weather application that displays current weather information for a selected location.

## Features
- Search for weather by city name
- Display current temperature
- Show weather conditions (clear, cloudy, rainy, etc.)
- Display humidity and wind speed
- Real-time weather data

## Files
- `index.html` - HTML structure
- `index.css` - App styling
- `index.js` - Weather API integration
- Weather icons (clouds.png, rain.png, snow.png, drizzle.png, mist.png)
- Additional icons (humidity.png, wind.png, search.png, clear.png)

## How to Use
1. Open `index.html` in your web browser
2. Enter a city name in the search field
3. Click the search button
4. View the current weather information
5. Check temperature, humidity, and wind speed
