# Drag & Drop Kanban Board

An interactive Kanban board application with drag and drop functionality for task management.

## Features
- Create tasks in different columns
- Drag and drop tasks between columns
- Organize tasks into categories (To-Do, In Progress, Done)
- Responsive design

## Files
- `index.html` - HTML structure
- `index.css` - Board styling
- `index.js` - Drag and drop functionality

## How to Use
1. Open `index.html` in your web browser
2. Create tasks in different columns
3. Drag tasks between columns to update their status
4. Organize your workflow visually
