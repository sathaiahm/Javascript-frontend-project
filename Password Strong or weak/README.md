# Password Strength Checker

A utility to check and display the strength of a password based on various criteria.

## Features
- Real-time password strength validation
- Visual feedback (weak, moderate, strong)
- Strength meter display
- Criteria checking (uppercase, lowercase, numbers, special characters)

## Files
- `index.html` - HTML structure
- `index.css` - Strength meter styling
- `index.js` - Password validation logic

## How to Use
1. Open `index.html` in your web browser
2. Enter a password in the input field
3. The strength checker will evaluate your password
4. View the strength indicator and feedback
5. Improve your password based on suggestions
