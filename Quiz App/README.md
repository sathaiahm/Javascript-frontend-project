# Quiz App

An interactive quiz application to test knowledge with multiple-choice questions.

## Features
- Multiple-choice questions
- Score tracking
- Timer for each question
- Progress indicator
- Results summary

## Files
- `index.html` - HTML structure
- `index.css` - Quiz styling
- `index.js` - Quiz logic and scoring

## How to Use
1. Open `index.html` in your web browser
2. Read each question carefully
3. Select your answer from the options
4. Click "Next" to proceed to the next question
5. View your final score at the end
