# Random Password Generator

A utility to generate random, secure passwords with customizable options.

## Features
- Generate random passwords
- Customize password length
- Include/exclude special characters
- Include/exclude numbers
- Include/exclude uppercase letters
- Copy to clipboard functionality

## Files
- `index.html` - HTML structure
- `index.css` - Generator styling
- `index.js` - Password generation logic
- `generate.png` - Generate button icon
- `copy.png` - Copy icon

## How to Use
1. Open `index.html` in your web browser
2. Set your password length
3. Select which character types to include
4. Click the "Generate" button
5. Click the "Copy" button to copy the password to clipboard
