# Notes App

A simple note-taking application to create, edit, and delete notes.

## Features
- Create new notes
- Edit existing notes
- Delete notes
- Persistent storage
- Clean and intuitive UI

## Files
- `index.html` - HTML structure
- `index.css` - Application styling
- `index.js` - Notes management logic
- `delete.png` - Delete icon
- `edit.png` - Edit icon
- `notes.png` - Notes icon

## How to Use
1. Open `index.html` in your web browser
2. Click "Add Note" to create a new note
3. Enter your note content
4. Click the edit icon to modify a note
5. Click the delete icon to remove a note
