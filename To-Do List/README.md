# To-Do List

A task management application to create, complete, and delete tasks.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Filter tasks (all, active, completed)
- Persistent storage
- Clean and simple interface

## Files
- `index.html` - HTML structure
- `index.css` - List styling
- `index.js` - Task management logic
- `checked.png` - Checked task icon
- `unchecked.png` - Unchecked task icon
- `icon.png` - App icon

## How to Use
1. Open `index.html` in your web browser
2. Type a task in the input field and press Enter
3. Click the checkbox to mark a task as complete
4. Click the delete icon to remove a task
5. Use filters to view different task statuses
